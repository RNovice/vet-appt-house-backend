# REST API application

## Install

```
npm install
```

## Run the app

```
npm run start
```

## Develop the app

```
npm run dev
```

# REST API

### Request

`GET /vetClinics/`

```
curl -i -H 'Accept: application/json' http://localhost:3000/vetClinics/
```
